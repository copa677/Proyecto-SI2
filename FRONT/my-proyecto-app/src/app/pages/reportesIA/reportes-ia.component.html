Consejos para usar comandos de voz:

  • Habla claramente y menciona el tipo de reporte que necesitas
  • Puedes especificar períodos: "último mes", "esta semana", "hoy", etc.
  • Menciona palabras clave: ventas, inventario, producción, clientes, bitácora
  • El sistema procesará automáticamente tu comando y generará el reporte
  • Si tu navegador no soporta voz, usa el campo de comando manual
